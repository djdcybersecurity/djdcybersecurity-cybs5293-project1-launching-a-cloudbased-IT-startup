# Team 1 – Montreal, Canada  
## Aurora Ledger – Fintech Cloud Platform

### 1. Company Overview
Aurora Ledger is a cloud-based Fintech startup headquartered in Montreal, Canada.  
The company specializes in providing **secure financial data processing**, **compliance automation**, and **API-based banking tools** to small and medium-sized financial institutions across North America.  

Montreal’s advanced cloud infrastructure, skilled tech workforce, and strong data privacy laws make it an ideal location to host Aurora Ledger’s primary data center operations.

---

### 2. Business Objective
Aurora Ledger aims to simplify the management of complex financial transactions by offering a centralized cloud solution that ensures:
- Automated compliance with Canadian and U.S. financial regulations.
- Encrypted data storage and real-time monitoring.
- Scalable API access for fintech developers and partner banks.

The project’s primary objective is to **design a secure and scalable cloud architecture** capable of handling sensitive financial data while maintaining cost efficiency.

---

### 3. Chosen Cloud Service Models
Aurora Ledger will use **two cloud service models** for implementation and comparison:

#### **PaaS (Platform as a Service)**
- Host a custom financial transaction engine and API gateway using **Proxmox VE** virtual machines.  
- Provide a platform for clients to build integrations, run analytics, and deploy their own compliance modules.

#### **SaaS (Software as a Service)**
- Deploy a **TurnKey Linux LXC container** with a pre-configured financial data management suite for direct client use.  
- Offer subscription-based access with secure login, data dashboards, and API access for automation.

---

### 4. Cloud Architecture Vision
Aurora Ledger’s architecture will integrate:
- **Secure data storage** using object storage and encryption at rest.
- **Load balancing and redundancy** to ensure high availability.
- **Logging and monitoring** for security events.
- **LocalStack/AWS compatibility** for scalable cloud testing and deployment.
